# ReSell Hub Server

## Purpose
Backend API for ReSell Hub marketplace with authentication, RBAC, products, orders, payments, wishlist, and dashboard stats.

##live url:
https://resell-hub-client-gamma.vercel.app/

https://resell-hub-server-rewt.onrender.com


## Features
- Express REST API
- MongoDB native driver
- Better Auth with MongoDB adapter
- RBAC middleware
- Product CRUD and admin moderation
- Order and payment APIs
- Stripe Checkout and webhook
- Wishlist APIs
- Dashboard analytics APIs

## Packages
- express
- mongodb
- dotenv
- cors
- cookie-parser
- better-auth
- @better-auth/mongo-adapter
- stripe
- nodemon

npm install
